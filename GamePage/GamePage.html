Simon
In Simon, you wll be shown a pattern of colors. You need to memorize this pattern and repeat it. The pattern starts short with just one color, and gets longer each round. If you mess up even once, the game ends, so watch closely.
Guess the Color
In this game, you wll be given an rgb code and you need to choose which color option that code matches up with. You only have three lives, so be careful and get as many correct as you can before you run out of lives.
